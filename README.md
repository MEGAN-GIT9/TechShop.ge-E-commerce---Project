# TechShop

[Visit page](https://megan-git9.github.io/TechShop.ge-E-commerce---Project/

## Description

TechShop is a modern single-page e-commerce web application developed using HTML, Tailwind CSS, and JavaScript. The goal of this project is to simulate the experience of shopping in a real online electronics store while demonstrating front-end web development concepts learned throughout the CS50 course.

The application allows users to browse a collection of electronic products, search for specific items, and filter products by category, brand, and price range. This makes it easy for users to quickly find products that match their needs. Each product has its own detailed page where users can view descriptions, specifications, pricing, discounts, and related products.

One of the main features of the project is the shopping cart system. Users can add products to their cart, update item quantities, remove products, and view the total order price before proceeding to checkout. The cart data is stored in the browser using LocalStorage, allowing users to keep their shopping cart even after refreshing the page.

The application also includes a simulated authentication system where users can register, log in, or use a Google login simulation. User information is stored locally to create a realistic front-end authentication experience without requiring a backend server.

The checkout page supports multiple payment methods, including bank card payments and installment payment simulation. Form validation is implemented to ensure users provide the required information before completing an order.

To improve the user experience, TechShop includes several additional features such as product search, dynamic filtering, recently viewed products, toast notifications, promotional banners, multilingual support (English and Georgian), and an AI Assistant interface that provides predefined responses to common customer questions.

Navigation is implemented as a Single Page Application (SPA) using hash-based routing. Instead of loading multiple HTML pages, the application dynamically displays different sections based on the current route, creating a smoother and faster browsing experience.

The project focuses on clean user interface design, responsive layouts, and interactive functionality. Tailwind CSS was used to build a modern and mobile-friendly interface, while JavaScript handles all application logic, including rendering products, filtering, shopping cart management, language switching, routing, authentication simulation, and checkout processing.

Overall, TechShop demonstrates how a fully interactive e-commerce website can be built using only front-end technologies. The project combines responsive design, dynamic content rendering, LocalStorage, DOM manipulation, and modern JavaScript techniques into a complete shopping experience that reflects many features found in real-world online stores.
